# Getting Started with FISCal

## Setup
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run Tests
```bash
python tests/fiscal_comprehensive_test.py
```

## Run Dashboard
```bash
streamlit run src/dashboard/streamlit_app.py
```

## Run CLI
```bash
python run_app.py
```
